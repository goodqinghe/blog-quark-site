5. 参考资料

写这篇笔记时，参考了以下非常优秀的文档和指南：

  • 中科大 Linux 201：SSH 篇

    强烈推荐，非常符合国内学生和开发者视角的系统性教程。

  • Arch Wiki：OpenSSH (简体中文)

    Linux 界的“圣经”，遇到配置文件不知道怎么写，或者 .ssh 权限问题，查它准没错。

  • A Practical Guide to SSH Tunnels (图解端口转发)

    把 -L 和 -R 端口转发原理解释得最好的一篇图文教程，非常生动。

  • DigitalOcean：SSH 基础配置指南

    保姆级的服务器安全配置教程，包含禁用 root 和禁用密码登录的最佳实践。

  • 如何优雅地使用 ssh config 文件

    详细列举了 ~/.ssh/config 中常用的参数和配置方法。

  • Ubuntu 官方：SSH 常见问题排查指南

    整理了各种连不上 SSH 的情况及排查思路。t
